# 🤖 Mini GPT — Lightweight Generative AI Assistant

Mini GPT is a lightweight generative AI assistant inspired by modern Large Language Models (LLMs).
The project focuses on building a simplified GPT-style conversational AI system with token generation, contextual understanding, streaming responses, and modular architecture.

This project was built to explore the internal architecture of modern AI assistants and understand how real-world conversational systems operate.

---

* 🧠 GPT-Style Conversational AI
* 💬 Real-Time Chat Interface
